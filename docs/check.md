# Описание Check

Описание check, как слоя в жизненом цикле запроса.

## Необходимость и ответственность
Check предоставляет возможность выполнять проверки перед выполнением основного запроса, на основе которых можно как пропустить так и запретить выполнять запрос дальше.

Check читает запрос, и используя его принимает решение имеет ли право этот запрос пройти дальше.

Чек ответственнен только за тот роут, к которому прикреплён и не более.
